## Aliases
Aliases can help make those tedious task just a bit easier.

To set aliases by adding to your rc files like such: `alias gs='git status`

### Git
    alias gs='git status '
    alias ga='git add '
    alias gb='git branch '
    alias gc='git commit'
    alias gp='git pull'
    alias gps='git push'
    alias gpsu='git push --set-upstream origin'
    alias gd='git diff'
    alias got='git '
    alias get='git '
